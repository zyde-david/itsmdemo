# IT Ticket System Demo

AI-powered IT ticket management system — Flask + SQLite + HTML/CSS/JS.

## Features

- 🎫 Ticket dashboard with filtering, status tracking, priority levels
- 👥 Staff management across 16 branches (3 provinces)
- 📦 IT asset inventory tracking
- 📚 Knowledge Base (ฐานความรู้ IT)
- 🔐 Admin panel with login
- 📊 Dashboard with stats & charts

## Quick Start

```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
# Admin password: demo2026
```

## Regenerate Demo Data

```bash
python generate_data.py
```

## Stack

- Python · Flask · SQLite
- Bootstrap 5 · DataTables · Chart.js
